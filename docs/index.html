Rcpp drat

This drat repository provides pre-releases and test builds for Rcpp and related packages.

Method One: First install the drat CRAN package via install.packages("drat") (or via install.packages("drat", repos="https://cloud.r-project.org")). Second add one line to your .Rprofile file (or to Rprofile.site): drat:::add("RcppCore"). That's it: install.packages() or update.packages() will now consider the drat repo along with the standard repo.

Method Two: Pass repo to install.packages(): install.packages("Rcpp", repos=c("https://RcppCore.github.io/drat", "https://cloud.r-project.org")). This also passes on the CRAN repo information in case other depencies are needed, one can add BioConductor or other repos as needed too. Works similarly for update.packages().

You can learn more about drat at the package documentation site which includes several vignettes.
